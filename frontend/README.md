# IntelliDocs — Frontend

Next.js (App Router) client for **IntelliDocs**. Full documentation, architecture, env vars, and deployment live in the **repository root**:

👉 **[`../README.md`](../README.md)** (open from repo root on GitHub)

## Quick commands

```bash
npm install
npm run dev
```

Configure `NEXT_PUBLIC_API_BASE_URL` in `.env.local` (see [`./.env.example`](./.env.example)).

## Production build

```bash
npm run build
npm start
```
